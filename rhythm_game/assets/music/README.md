# Music Assets

## Supported Formats
- MP3 (.mp3) - Recommended
- OGG (.ogg) - Better compression
- WAV (.wav) - Uncompressed quality

## Adding Your Music
1. Place your music files in this directory
2. Update the song path in your chart files
3. Supported formats: MP3, OGG, WAV

## Demo Track
The game includes a procedurally generated demo track that plays when no music file is found.

## Recommended Specifications
- Sample Rate: 44.1kHz
- Bit Rate: 128kbps or higher (for MP3)
- Channels: Stereo

## File Naming Convention
Use descriptive names:
- `song_title_artist.mp3`
- `demo_track_01.ogg`
- `electronic_beat.wav`
